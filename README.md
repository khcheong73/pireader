# pireader
Raspberry Pi Zero W as Barcode Reader 

Components
- RaspberryPi Zero W
- Nokia 5110 LCD (SPI)
- Waveshare Barcode Scanner module (https://www.waveshare.com/barcode-scanner-module.htm)
